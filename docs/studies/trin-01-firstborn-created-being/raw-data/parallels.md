# Cross-Testament Parallels

- Col 1:15 → Rom 8:29 (nt, vocabulary (firstborn, image) — both use prototokos and eikon)
- Col 1:15 → Heb 1:6 (nt, vocabulary (firstborn) — both use prototokos of Christ)
- Col 1:15 → Rev 1:5 (nt, vocabulary (firstborn) — both use prototokos of Christ + 'dead')
- Col 1:15 → 2 Cor 4:4 (nt, vocabulary (image) — 'image of God' applied to Christ)
- Col 1:15 → Gen 1:26 (ot, vocabulary (image) — 'image of God' creation language)
- Col 1:18 → Rev 1:5 (nt, vocabulary + structural — prototokos from/of the dead + faithful witness)
- Col 1:18 → Heb 12:23 (nt, vocabulary — 'church' + 'firstborn' in same verse)
- Col 1:18 → Eph 1:22 (nt, structural — 'head' + 'church' + 'all things')
- Rev 3:14 → 2 Pet 3:4 (nt, vocabulary — 'beginning' + 'creation' used together)
- Rev 3:14 → Rev 1:5 (nt, vocabulary + structural — 'faithful witness' in both; same Christological title set)
- Rev 3:14 → Col 4:16 (nt, vocabulary — 'church' + 'Laodicea' — same congregation addressed)
- John 1:3 → Col 1:16 (nt, vocabulary + theological — 'all things' created 'through' Christ)
- John 1:3 → 1 Cor 8:6 (nt, vocabulary + theological — 'all things through him' creation formula)
- John 1:3 → Heb 2:10 (nt, vocabulary — 'all things through him')
- John 1:3 → Rom 11:36 (nt, vocabulary + structural — threefold creation formula (of/through/to))
- Heb 1:6 → Psa 89:27 (ot, theological — firstborn applied to Davidic figure; angelic context)
- Col 1:16 → Gen 1:1 (ot, vocabulary — 'heaven' + creation language)
- Col 1:16 → Gen 1:26 (ot, vocabulary — 'dominion' + 'heaven' — creation rulership)
